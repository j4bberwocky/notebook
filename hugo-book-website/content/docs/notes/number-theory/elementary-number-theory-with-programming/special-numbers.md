# Special Numbers: Triangular, Oblong, Perfect, Deficient and Aboundant

{{< hint info >}}
**Triangular Number**  
The triangular numbers are given by the following explicit formulas

{{< katex display >}}
T_n = \sum_{k=1}^{n}k = 1 + 2 + \dots + n = \frac{n(n + 1)}{2}  
{{< /katex >}}
